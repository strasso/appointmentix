# Appointmentix Framer Nativ-Anleitung

Das ist die nativ editierbare Version (kein Embed-Zwang).

## Dateien in diesem Ordner

- `FRAMER-STYLES.md` (Design-Tokens)
- `FRAMER-SECTIONS.md` (Layout-Struktur)
- `FRAMER-COPY-DE.md` (alle deutschen Texte)
- `FRAMER-FORM-LOGIK.md` (Form-Verhalten + Calendly)

## Schritt-für-Schritt in Framer

1. Neue Seite `Start` erstellen.
2. Neue Seite `book-a-call` erstellen.
3. Farb- und Textstile aus `FRAMER-STYLES.md` anlegen.
4. Auf `Start` die Sections aus `FRAMER-SECTIONS.md` aufbauen.
5. Texte 1:1 aus `FRAMER-COPY-DE.md` einsetzen.
6. Auf `book-a-call` Formular aufbauen (Labels aus `FRAMER-COPY-DE.md`).
7. Formular-Weiterleitung nach `FRAMER-FORM-LOGIK.md` einrichten.
8. Buttons `App bauen` auf `/book-a-call` verlinken.

## Ergebnis

Du hast danach eine visuell in Framer editierbare Appointmentix-Seite
im gewünschten creme/beige/braun/ocker Stil.
