# Appointmentix Framer Styles (Nativ)

## 1. Farb-Styles in Framer anlegen

Lege in Framer unter `Styles -> Colors` diese Tokens an:

- `Sand / 50`: `#F4EDE2`
- `Sand / 100`: `#EFE4D5`
- `Surface / Base`: `#FFF9F0`
- `Ink / Primary`: `#2F2218`
- `Ink / Muted`: `#6F5D4A`
- `Brand / Primary`: `#8A5A2F`
- `Brand / Secondary`: `#9E6A3B`
- `Night / 900`: `#16131D`
- `Night / 800`: `#221B2A`
- `Text / Light`: `#F5E6CF`
- `Text / Light Muted`: `#CEBDA2`
